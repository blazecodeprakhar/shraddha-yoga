# Shraddha Yoga & Breathwork

A premium, high-performance single-page website for **Shraddha Yoga & Breathwork** coaching. Built with semantic HTML5, modern vanilla CSS, responsive layouts, and interactive breathing guidance.

## Features
- **Interactive Breath Trainer**: A dynamic breathing ring with visual guides and interval cycles (4s inhale / 4s exhale).
- **FAQ Accordion**: Seamless expand/collapse panel for client questions.
- **Weekly Rhythm Schedule**: Structured weekly offerings timetable.
- **Clean & Premium Aesthetic**: Harmonious warm dark palette, sophisticated typography (Google Fonts Outfit & Playfair Display), and refined spacing.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile views.
- **Lightweight**: Pure HTML/CSS/JS without heavy framework dependencies.

## Tech Stack
- HTML5
- CSS3 (Vanilla Custom Properties & Grid/Flexbox Layouts)
- JavaScript (ES6+)
